## 🚀 如何使用一键部署

### 超级简单部署（3步）

```
步骤1：双击 deploy.bat 或右键 deploy.ps1 选择"使用PowerShell运行"
步骤2：输入 PostgreSQL 密码
步骤3：输入 Y 两次（自动生成密钥）
完成！访问 http://localhost:5000
```

**就是这么简单！** 🎉

## 📋 部署脚本自动完成的任务

1. ✅ 检查 Node.js 是否安装
2. ✅ 检查 pnpm 是否安装（未安装则自动安装）
3. ✅ 检查 PostgreSQL 是否运行（未运行则自动启动）
4. ✅ 安装项目依赖（约5-10分钟）
5. ✅ 创建 `.env` 配置文件
6. ✅ 自动生成安全的密钥
7. ✅ 创建数据库
8. ✅ 运行数据库迁移
9. ✅ 提示启动应用
